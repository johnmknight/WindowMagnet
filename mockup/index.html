WindowMagnet — UI mockup

A compact picker window on monitor 2. Tiny live thumbnails of every window buried behind the fullscreen game on monitor 1 — click one to recall it. The game keeps focus.

[idle]Picker enumerating windows — click any thumbnail to recall.
Static HTML mockup. The picker is roughly to scale — about 360px wide on a 1080p second monitor (~19% of screen width). The real app uses Win32 EnumWindows, the DWM Thumbnail API for live previews, and SetWindowPos with SWP_NOACTIVATE for focus-preserving recall. See DESIGN.md.
